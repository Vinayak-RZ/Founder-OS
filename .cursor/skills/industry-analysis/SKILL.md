# Industry analysis

## When to use
- Industry sizing, segments, and growth drivers
- Value chain and buyer personas (Indian SME manufacturers)
- Competitive landscape and differentiation
- Industrial pain points (energy, OEE, data fragmentation)

## Output shape
1. **Scope** — geography, segment, time horizon
2. **Market structure** — key players, substitutes, barriers
3. **Pain & budget** — operational KPIs affected (₹ impact where possible)
4. **Implications** — 3–5 bullets for product/GTM

## Founder OS role
Run deep analysis in Cursor with this skill. Ingest final memos into the world's `industry/` or `market/` vault folder, then query via Vault specialist.
